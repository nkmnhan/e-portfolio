# E-Portfolio

A professional portfolio platform for **Concept Artists** and **Film Animation Developers**.

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS + clsxMerge
- **Animation**: Framer Motion
- **3D**: React Three Fiber

## Quick Start

```bash
cd webs/portfolio-nextjs
pnpm install
pnpm dev
```
